1. *alias*

```
$ vi ~/.bashrc

#no space #esc and :wq (write and quite)
alias gcptest1="ssh -i ~/.ssh/gcp jk@35.196.84.132"

$ source ~/.bashrc

#to access GCP server via terminal
$ ssh -i ~/.ssh/gcp jk@35.196.84.132

# or use alias
$ gcptest1

```
