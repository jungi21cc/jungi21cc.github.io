**HTML**\

The most important components of HTML is tag. Let's see the most frequent usages of tags:

*a*
>is anchor to use hyperlink

```HTML
<p>
    link: <a href="http://www.google.com">google</a>
</p>
```

*body*
>is main contents in HTML

```html
<!DOCTYPE html>
<html>
  <head>
    ...skip...
  </head>
  <body>
    ...this is body paragraph...
  </body>
</html>
```
