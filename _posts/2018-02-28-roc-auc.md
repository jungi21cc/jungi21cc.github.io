### ROC & AUC

### 1. *Confusion matrix*

```python
from sklearn.metrics import confusion_matrix

$ y_true = [2, 0, 2, 2, 0, 1]
$ y_pred = [0, 0, 2, 2, 0, 2]
$ confusion_matrix(y_true, y_pred)
array([[2, 0, 0],
       [0, 0, 1],
       [1, 0, 2]])
```
